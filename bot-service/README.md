# README
## 配置环境变量
```
export DB_URL="jdbc:mysql://124.70.222.207:3306/unigpt"
export DB_USERNAME="xxx"
export DB_PASSWORD="xxx"
```
## 运行
Linux 和 MacOS:
```
./mvnw spring-boot:run
``` 
Windows:
```
./mvnw.cmd spring-boot:run
```
后端URL为http://localhost:8080
## 构建
Linux 和 MacOS:
```
./mvnw package
```
Windows:
```
./mvnw.cmd package
```
